Abstract

Background: High‑dose tablets (≥90% API) face capping and lamination risks due to poor compressibility. Traditional QbD requires 50–200 experiments — costly and slow.

Methods: A hybrid AI framework integrates a Physics‑Informed Neural Network (PINN) (embedding Heckel equation & Elastic Failure Risk Function) with NSGA‑II multi‑objective optimization. Trained on 45 in‑silico formulations (LHS).

Results: PINN achieved R² = 0.96 ± 0.02, RMSE = 0.08 MPa. Outperformed MLP, RF, XGBoost (p < 0.05). Identified 90.5% paracetamol formulation satisfying σₜ ≥ 2 MPa and EFRF < 0.5.

Conclusions: Physics‑guided AI enables reliable digital design spaces for Pharma 4.0. Experimental validation is ongoing.

Pareto‑Optimal Design Space

The NSGA‑II optimization generated a robust Pareto front. The optimal formulations cluster at ~90.5% paracetamol loading, satisfying both mechanical constraints (σₜ ≥ 2 MPa and EFRF < 0.5). Formulations exceeding this threshold show a steep rise in predicted capping risk.

Core Code Snippet (PINN Loss)

# Hybrid loss: L = ω₁ · MSE_data + ω₂ · MSE_physics class PINN(nn.Module): def compute_loss(self, X, y_true, omega_data=0.8, omega_physics=0.2): y_pred = self.forward(X) data_loss = F.mse_loss(y_pred, y_true) # Physics residuals: Heckel equation & EFRF physics_loss = self.physics_loss(X, y_pred) total_loss = omega_data * data_loss + omega_physics * physics_loss return total_loss, data_loss, physics_loss
